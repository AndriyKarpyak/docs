  10. case class HistoryConfig(layer: Name, environment: EnvironmentConfig, statistics: StatisticsConfig) extends Product with Serializable

    Configuration controlling how the history is published in the output catalog.

    Configuration controlling how the history is published in the output catalog. History is published in a generic layer and it is made of job information (which input catalogs and which version were used) and compilation statistics. This is fairly expensive operation, so the results should be cached locally.

    layer

    the layer name for the history
