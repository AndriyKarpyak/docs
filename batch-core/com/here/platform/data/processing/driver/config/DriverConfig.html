com.here.platform.data.processing.driver.config

DriverConfig

Related Doc: package config

case class DriverConfig(appName: String, history: HistoryConfig, parallelUploads: Int, parallelRetrievers: Int, numCommitPartitions: Int, sparkStorageLevels: SparkStorageLevelsConfig, state: StateConfig, schema: SchemaConfig, disableIncremental: Boolean) extends Product with Serializable

Configuration needed to instantiate and configure a com.here.platform.data.processing.driver.Driver.

appName

the name of the application to set in the Spark context

history

the configuration of history publishing

parallelUploads

the number of parallel uploads performed by the framework internally, like in data and state publishing

parallelRetrievers

the number of parallel retrieves performed by the framework internally, like in data and state retrieving

numCommitPartitions

the maximum number of partition to commit to in the multipart commit of DataStore

sparkStorageLevels

the spark storage levels configuration

state

the configuration for state

schema

the protobundle schemas publishing configuration

disableIncremental

disable incremental compilation

Note

numCommitPartitions has to be smaller than 100 due to DataStore API specification
