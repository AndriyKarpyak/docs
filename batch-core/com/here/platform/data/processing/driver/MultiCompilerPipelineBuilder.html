MultiCompilerPipelineBuilder

Related Doc: package driver

class MultiCompilerPipelineBuilder extends AnyRef

Builds a multi compiler based mode by getting compilers one by one.

Note

the class DriverBuilder offers the method DriverBuilder.newMultiCompilerPipelineBuilder to create this builder, it is suggested to use it

,

executor ids can be supplied or can be left out so that they are auto-generated. When supplied they must be unique or an execption will be thrown at runtime. When they are automatically generated they have the form of pipelineId-n where n is the index of the position of the executor in the constructed pipeline sequence

,

in case a single NonIncrementalCompilerExecutor is added then the mode is created as non incremental on the contrary the mode is created as incremental
