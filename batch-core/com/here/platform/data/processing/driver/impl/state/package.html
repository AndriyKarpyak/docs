    Serializes the compiler state represented as RDD[K, V] to/from an RDD[Payload].

    Serializes the compiler state represented as RDD[K, V] to/from an RDD[Payload]. The compiler-specific serializer for a single spark partition needs to be defined when instantiating class. This class save tha state in a deterministic way by:

    - partitioning by key - sorting by key inside the partitions

    K

    the type of the state key, to write the state in a stable way this type is supposed to have an ordering implicitly defined

    V

    the type of the state values
