case class ToProcessConfig(catalogHrn: Option[String] = None, update: Option[ToProcess] = None) extends Product with Serializable

Helper class for parsing of com.here.platform.data.processing.driver.job.InCatalog classes with partial content as part of JobConfig. In this class, all parameters are optional to allow this partial filling from one configuration source, and merging multiple sources (for example configuration file, command line) to get all required parameters for a com.here.platform.data.processing.driver.job.InCatalog instance.
