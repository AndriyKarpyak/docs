case class JobConfig(catalogHrn: Option[String] = None, catalogVersion: Option[BaseVersion] = None, catalogsToProcess: Map[String, ToProcessConfig] = Map(), commitToken: Option[String] = None) extends Product with Serializable

Helper class for parsing partial data of a com.here.platform.data.processing.driver.job.Job object.

This class is used by the framework to parse the runtime configuration from multiple sources and the create the com.here.platform.data.processing.driver.job.Job instance that actually drives the compilation.

Do not use this class directly, if not to parse configuration from JSON or similar.

In this class, all parameters are optional to allow partial filling from one configuration source, and merging multiple sources (for example configuration file, command line) to get all required parameters for a com.here.platform.data.processing.driver.job.Job.

catalogHrn

HRN of the DataStore catalog to publish to

catalogVersion

the version that this filter's DataStore was at when this job changeset was created

catalogsToProcess

the set of upstream filters to use for processing, mapped from com.here.platform.data.processing.catalog.Catalog.Id to ToProcessConfig

commitToken

multipart commit token

Note

For unit test, synthesize a com.here.platform.data.processing.driver.job.Job class directly with the parameters needed for the tests
