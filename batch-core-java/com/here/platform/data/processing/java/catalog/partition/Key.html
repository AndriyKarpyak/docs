com.here.platform.data.processing.java.catalog.partition

Key

Related Doc: package partition

class Key extends Wrapper[catalog.Partition.Key]

A key to uniquely identify a partition in compilers, pipelines and, in general RDDs.

As it contains the identifier of the corresponding catalog, partition keys of multiple catalogs can be processed in the same RDD without collisions. Output catalogs have also their unique id, so it is in theory possible to mix input and output partitions in the same RDD.

Note

This is a Java friendly wrapper of com.here.platform.data.processing.catalog.Partition.Key.
